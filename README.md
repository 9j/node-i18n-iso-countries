#i18n-iso-countries

i18n for ISO 3166-1 country codes

## usage

`````javascript

var countries = require("i18n-iso-countries");
console.log("US => " + countries.getName("US", "de")); // US => Vereinigte Staaten von Amerika

`````

## supported languages

* `de`: german
